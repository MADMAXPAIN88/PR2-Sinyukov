# Для чего нужен файл README.md в репозитории?
Главный документ проекта, служащий его «лицом» и кратким руководством. 
Он нужен для объяснения сути проекта, его ценности, инструкции по установке,
запуску и использованию

# Какую информацию обычно размещают в файле README.md
обычно содержит название, краткое описание целей проекта,
инструкции по установке, запуску, использованию, 
примеры кода, а также раздел о лицензии и участниках

# Как создать заголовок первого уровня в Markdown?
Использовать "#"

# Как создать нумерованный список в Markdown?
написать цифру после точку пробел и текст(пример:1. Первый пункт)

# Для чего нужна вики-страница в репозитории?
Предназначена для хранения развернутой документации проекта, инструкций, правил участия
